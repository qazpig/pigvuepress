# 矮人礦坑：緊張刺激的寶石爭奪戰

矮人礦坑是一款輕鬆有趣的派對遊戲，玩家們化身為尋寶的矮人，在危機四伏的礦坑中競相搜尋寶石。

## 遊戲機制

1. 回合制抽牌，決定是否繼續探索或返回營地。
2. 風險與回報的權衡，增加遊戲的刺激感。
3. 簡單的規則設計，適合各年齡層的玩家。

矮人礦坑的魅力在於其簡單yet深具策略性的玩法。每個決定都可能導致豐厚的回報或失去所有，這種緊張感使得每局遊戲都充滿樂趣。它是快速遊戲之夜或作為更長遊戲之間的調劑的理想選擇。
